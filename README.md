# Aplicação Todo List em JavaScript
Esta é uma aplicação simples de lista de tarefas construída usando HTML, CSS e JavaScript. A aplicação permite que você crie, atualize e delete tarefas que você precisa completar.

![image](https://user-images.githubusercontent.com/62481734/221416821-e67e8a3f-85de-40e9-8687-a265da893df2.png)


# Funcionalidades
Adicionar uma nova tarefa
Marcar uma tarefa como concluída
Excluir uma tarefa

# Instalação
Para usar a aplicação Todo List, siga estes passos:

Clone o repositório usando git clone https://github.com/LuanPcode/todo_list.
Abra o arquivo index.html no seu navegador.
Utilização
Para adicionar uma nova tarefa, basta digitar a descrição da tarefa no campo de entrada e pressionar a tecla Enter.

Para marcar uma tarefa como concluída, clique na caixa de seleção ao lado da descrição da tarefa.

Para excluir uma tarefa, clique no botão "X" ao lado da descrição da tarefa.
